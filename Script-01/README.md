# My Bash Scripting Notes 👾

Hey there! This is my personal space where I’m keeping all my notes and scripts as I learn Bash scripting from scratch. 

My big goal is to go from a complete beginner to being able to write actual automation scripts used in DevOps and SRE jobs (like managing logs, backups, and servers).

---

## 🗺️ What I'm Learning (My Roadmap)

### 🟢 1. The Basics (What I'm doing today!)
*   **If/Else Conditions:** Making scripts make decisions using `if`, `else`, and `elif`.
*   **Comparison Operators:** Learning how to compare numbers and text (using weird-looking things like `-eq`, `-ne`, `-gt`, `-lt`).
*   **Loops:** Making things run over and over using `for`, `while`, and `until`.

### 🟡 2. Getting Advanced
*   **Functions & Arrays:** Keeping my code clean and handling lists of data.
*   **Script Arguments:** Passing data into scripts when running them (like `$1`, `$2`, `$@`).
*   **Error Handling:** Making sure my scripts don't just crash when something goes wrong.
*   **File Processing:** Reading and playing with text files automatically.

### 🔴 3. Real DevOps Stuff (The Ultimate Goal)
Once I know the basics, I want to build real scripts for:
*   🧹 Cleaning up old server logs automatically.
*   💾 Making automated backups.
*   🐳 Setting up Docker & Kubernetes stuff.
*   📊 Monitoring server health (CPU/Memory) and sending alerts.
