# ATM System (Bash)

A simple ATM System built using Bash scripting while learning Linux and Bash fundamentals.

## Features

* Check Balance
* Deposit Money
* Withdraw Money
* Exit Program

## Concepts I Practiced

* Variables
* User Input (`read`)
* Output (`echo`)
* `if`, `elif`, `else`
* `while` Loop
* Arithmetic Operations
* Basic Input Validation

## How to Run

```bash
chmod +x atm.sh
./atm.sh
```

## What I Learned

While building this project, I learned how to:

* Create interactive Bash scripts
* Build a menu using a `while` loop
* Make decisions using `if` and `elif`
* Update variables using arithmetic operations
* Validate user input
* Solve problems using programming logic

This is one of my beginner Bash scripting projects. I will continue improving it by adding functions, file handling, login, and transaction history as I learn more.
